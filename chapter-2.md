### 局域网大文件

偶尔可能在局域网特殊机器上会传一些大文件，ssh 也可能无法使用

临时开启 http server 是一个非常使用的方法

::: code-group

```bash [Python]
python3 -m http.server 8888
```

```bash [NodeJS]
npx serve -p 8888
```

```bash [PHP]
php -S localhost:8888
```

```bash [Ruby]
gem install webrick
ruby -run -e httpd . -p 8888
```
:::
